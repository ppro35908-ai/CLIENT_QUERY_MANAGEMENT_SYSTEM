# CLIENT_QUERY_MANAGEMENT_SYSTEM

##Description
In this project, we created client management dashboard by using Language:python, Database:postgresql, Visualisation tool :Streamlit and the Libraries: Pandas, datetime, psycopg2. In this dashboard, client can raise their queries and that should be solved by support/admin team immediately by using realtime dashboard.

##Features
Add client queries
Update query status
Store client information
User login system

##Tech Stack
Python
PostgreSQL
Streamlit
Libraries:Pandas, datetime, psycopg2
